Detailed Description

Execution context managed by the Host.

The Host MUST pass the pointer to the execution context to evmc_execute_fn. The EVM MUST pass the same pointer back to the Host in every callback function. The context MUST contain at least the function table defining the context callback interface. Optionally, The Host MAY include in the context additional data.
