Detailed Description

The union representing evmc_result "optional data".

The evmc_result struct contains 24 bytes of optional data that can be reused by the object creator if the object does not contain evmc_result::create_address.

An EVM implementation MAY use this memory to keep additional data when returning result from evmc_execute_fn. The host application MAY use this memory to keep additional data when returning result of performed calls from evmc_call_fn.

See also
evmc_get_optional_data(), evmc_get_const_optional_data().
